# Shelly Sharma Academy Blueprint & Integration Documentation

Welcome to the official digital blueprint and integration documentation for **Shelly Sharma Academy** (`www.shellysharma.co.in`).

## 1. Educational Blueprint

Shelly Sharma Academy is a premier English educational academy tailored for ambitious students, working professionals, and global test takers. Our curriculum is designed to transform hesitation into eloquent spoken and written English.

### Core Curriculums
- **Elite Spoken English Mastery**: Intensive training targeting professional presentations, dynamic conversation patterns, confidence building, and advanced fluency.
- **Advanced Grammar & Syntax Clinics**: Focused workshops on structural precision, cohesive writing styles, business correspondence, and complex sentence mechanics.
- **IELTS Academic/General Prep**: Advanced guidance designed to achieve Band 8+ scores with targeted strategy modules in listening, speaking, reading, and writing.

---

## 2. Platform Architecture

### Tech Stack
- **Frontend**: React 18, Vite, TypeScript, and Tailwind CSS.
- **Animations**: Framer Motion (`motion/react`) for refined transition kinetics.
- **Authentication**: Firebase Auth with Google OAuth provider.
- **Email Delivery**: Client-side Gmail API (`gmail.send` scope) proxy for instant, automated dual emails (Student Confirmation & Administrator Alert).

---

## 3. SEO & Discovery Blueprint

- **Primary Domain**: `https://www.shellysharma.co.in`
- **Canonical Configuration**: Enforced globally on all HTML views to concentrate domain authority.
- **Robots.txt Location**: `/robots.txt` (References the primary sitemap).
- **Sitemap Location**: `/sitemap.xml` (Includes priorities for primary anchor links).
- **Open Graph (OG) Protocols**: Built-in preview parameters optimized for LinkedIn, WhatsApp, Facebook, and Twitter.
- **Structured Data (JSON-LD)**: Rich schema tags embedded in the root index file to register the Academy, Local Business, and Educational Program with Google Knowledge Graph.
